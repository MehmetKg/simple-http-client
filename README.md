# Simple HTTP Client

Bu proje, Python'un `socket` modülünü kullanarak en temel düzeyde bir HTTP istemcisi nasıl yazılır sorusuna yanıt vermek amacıyla oluşturulmuştur.

## Özellikler

- TCP protokolü üzerinden bağlantı kurar.
- Bir HTTP GET isteği gönderir.
- Sunucudan gelen yanıtı terminalde gösterir.
- Bağlantı tamamlanana kadar tüm verileri alır.
